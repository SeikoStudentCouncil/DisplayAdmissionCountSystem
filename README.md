# DisplayAdmissionCount

## ビルド

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate
```

## 仕様説明

### 目的
展示団体に入る人を監視し入場時間や室内にいる人数をもに制限する。
